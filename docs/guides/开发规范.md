## 代码结构

---

- **UI 层**：`ui` 包 — Activity、Dialog、自定义 View、RecyclerView 适配器等
- **工具层**：`util` 包 — 通用工具与辅助方法
- **调试**：`ui/test` 包 — 调试面板相关（`TestManager`、`TestModeDialog` 等）

### 命名规范

- **类名**：PascalCase，如 `MainActivity`、`TtsEngineHelper`
- **方法名 / 变量名**：camelCase，如 `startSpeak()`、`isEngineAvailable`
- **常量名**：UPPER_SNAKE_CASE，如 `AUDIO_FILE_NAME`、`MAX_RETRY_COUNT`
- **包名**：全小写，如 `com.citadawn.speechapp.util`
- **资源文件**：下划线分隔，如 `activity_main.xml`、`btn_primary_bg.xml`

### 代码注释

- 公共类和方法添加 JavaDoc
- 复杂业务逻辑添加行内注释
- 使用 `// region ... // endregion` 分区

### 颜色与尺寸

- **仅 1 次**：可硬编码（IDE 可能提示，不阻止构建）
- **2 次及以上**（或预计会复用）：使用 `colors.xml` 中的 `@color/*` 或 `dimens.xml` 中的 `@dimen/*`
- **相同色值 / 数值只保留一个资源名**，禁止在 `colors.xml` / `dimens.xml` 里重复起名

用 Android Studio「Find Usages」（Alt+F7）统计引用次数。

- **运行时设置字体大小**：

```java
// 推荐方式：在布局中定义
android:textSize="@dimen/sp_16"

// 运行时设置（如必须）
textView.setTextSize(TypedValue.COMPLEX_UNIT_PX,
    getResources().getDimension(R.dimen.sp_16));
```

### 语言列表去重

项目在显示 TTS 语言列表时会自动去除显示名称重复的语言：

- **去重原则**：当多个 `Locale` 对象（如 `zh-Hans-CN` 和 `zh-CN`）显示名称相同时，只保留一个
- **优先级规则**：
  1. 默认语言始终保留
  2. 优先保留有更多发音人的 Locale
  3. 发音人数量相同时，优先保留更简洁的 Locale（如 `zh-CN` 优先于 `zh-Hans-CN`）
- **实现位置**：`TtsLanguageVoiceHelper.sortLocalesByDisplayName()` 方法

### 信息图标规范

项目使用统一矢量图 `@drawable/ic_info`（Material 信息图标）。引用处用布局设尺寸、`app:tint` / `app:iconTint` 设颜色：

- **页面内容区域**：`14dp`，`app:tint="@color/hint_gray"`，与文字紧挨（`marginStart` 默认 0dp），须用 `InfoIconPositionHelper` 动态定位到标签右上角
- **顶部栏菜单**：`app:iconTint="@color/pure_white"`（菜单项由系统按 action bar 尺寸显示）

**动态定位实现**：

```java
// 在 setupInfoIconPositions() 中添加
findViewById(android.R.id.content).post(() -> {
    TextView tvLabel = ViewHelper.findTextViewByText(this, getString(R.string.your_label));
    ImageView ivInfo = findViewById(R.id.ivYourInfo);
    if (ivInfo != null && tvLabel != null) {
        InfoIconPositionHelper.setIconPosition(ivInfo, tvLabel);
    }
});
```

动态定位让图标偏移到文字高度的 60% 位置，像上标一样对齐。

**布局模板**：

```xml
<ImageView
    android:id="@+id/ivYourInfo"
    android:layout_width="14dp"
    android:layout_height="14dp"
    android:layout_marginStart="1dp"
    android:src="@drawable/ic_info"
    app:tint="@color/hint_gray"
    android:contentDescription="@string/your_info_title"
    android:clickable="true"
    android:focusable="true" />
```

## 国际化规范

---

### 字符串资源

用户可见文本**一律**使用 `@string/*`（与「颜色与尺寸」的 2 次规则不同：字符串**无论几次都必须**进资源，以便翻译）。

- 中文：`values-zh/strings.xml`（默认）；英文：`values/strings.xml`
- 各语言资源 **ID 保持一致**；先写中文，再同步英文

### 命名规范

- 使用描述性名称，如 `tts_engine_not_available` 而非 `error_msg`
- 按功能模块分组，如 `tts_*`、`ui_*`、`error_*`
